# 표·차트 생성기 & 그라디오 UI

> 2025 상반기 **표·차트 생성기** 개발과 **AI 모델 데모용 그라디오 UI** 개발·모델 적용 요약.  
> **출처**: 2025 성과아카이브, Jira [AI-821](https://miridih.atlassian.net/browse/AI-821), [AI-741](https://miridih.atlassian.net/browse/AI-741)

---

## 1. 표·차트 생성기 (AI-821)

| 구분 | 내용 |
|------|------|
| **목적** | 표·차트를 자동 또는 반자동으로 생성하는 도구/서비스. (상세 스펙은 팀·제품 요구사항 따름.) |
| **내 역할** | 설계·구현·연동. |
| **참고** | [AI-821](https://miridih.atlassian.net/browse/AI-821) |

---

## 2. 그라디오 UI (AI-741)

| 구분 | 내용 |
|------|------|
| **목적** | **AI 모델 데모**를 위한 **Gradio** 기반 UI 개발 및 **모델 적용**. |
| **내 역할** | Gradio UI 설계·구현, 모델 엔드포인트 또는 추론 로직 연동. |
| **참고** | [AI-741](https://miridih.atlassian.net/browse/AI-741) |

---

## 3. 사용한 툴·기술

- **표·차트 생성기**: (구현 방식은 AI-821·코드베이스 참고.)
- **그라디오 UI**: Python, **Gradio**, AI 모델 API 또는 로컬 추론.

---

## 4. 이력서·이직 활용

- **키워드**: 표/차트 생성, AI 데모, Gradio, 모델 연동.
- 상세 스토리가 필요하면 Jira AI-821·AI-741 및 Confluence에서 “표·차트 생성기”“그라디오” 검색해 보강 가능.
