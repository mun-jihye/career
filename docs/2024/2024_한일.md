## 1. 2024년 핵심 성과 (DX 개발)

### 1.1. [데이터 파이프라인 · 내부 도구(Admin)] 레이아웃 자동 생성 모델용 학습 데이터 생성기 (xml-image-csv-converter)

| 구분 | 내용 |
|------|------|
| **JD 매핑** | **데이터 파이프라인** — 디자인 온톨로지 데이터(XML)를 AI 학습용 이미지·CSV로 변환하는 파이프라인 설계·개발. **내부 도구(Admin/CMS)** — AI Researcher가 직접 사용할 수 있도록 모듈화한 도구 개발 |
| **성과** | 레이아웃 자동 생성 모델 연구에 필요한 **학습용 데이터 생성기 (xml2image-csv converter)** 개발 및 전달 |
| **기여도 / 내 역할** | • 요구사항에 맞는 **데이터 생성 파이프라인 설계·개발** (XML → Image, CSV 변환) <br>• AI Researcher·유관 부서가 쉽게 이용할 수 있도록 **모듈화** 구현 — **사용자 문제 해결 중심 프로덕트 지향성** <br>• Save Process(xml/idx → image·csv), Visualize Process(브라우저 미리보기), Element Image 추출 설계·구현 <br>• CSV 포맷(template metadata)·SIMPLE_TEXT 등 XML 구조 반영 문서화, 배포·브랜치 이력 정리 |
| **기술 스택** | XML 파싱, Puppeteer(렌더링), CSV 입출력, Python·Node 연동, CLI 모듈화, Docker·EC2·S3·ECR |
| **상세 문서** | **[xml-image-csv-converter.md](./xml-image-csv-converter.md)** — 아키텍처, Process(Save/Visualize/Element 추출), CLI 명령어, CSV 포맷, SIMPLE_TEXT 속성, 배포·개발 이력 |
| **비고** | Confluence: [xml-image-csv-converter 1.0](https://miridih.atlassian.net/wiki/spaces/AIDMLOps/pages/584451340), [개인 스페이스 폴더](https://miridih.atlassian.net/wiki/spaces/~712020a23635f8190743df9448a03c47cdb090/folder/713565403) |

- **나 → 리드 (기대·강점)**  
  명확한 비전·방향 제시, 지속적 피드백·인정. 스쿼드 분리 후에도 관심·지원, DinoV2 개발 시 GPU 설정 등 적극적 지원에 대한 만족.
